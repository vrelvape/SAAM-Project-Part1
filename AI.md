Use of Large Language Models (LLMs)

Generative AI tools (ChatGPT, OpenAI) were used as a support tool throughout the project. Their use was limited to assistance with coding tasks (e.g., debugging error messages, clarifying Python or NumPy functions, and improving code structure), as well as to improve the clarity and organization of written materials (e.g., README formatting and language refinement).

In particular, LLMs were used to:
	•	help diagnose and resolve technical errors encountered during implementation;
	•	suggest improvements in code modularization and readability (e.g., structuring the project into a src architecture);
	•	clarify certain financial or statistical concepts when needed.

However, all key methodological choices—including data cleaning procedures, construction of the investment universe, definition of the minimum variance optimization problem, implementation of the backtesting logic, and interpretation of the empirical results—were developed and validated independently by the group.

The coding of the final pipeline, the generation of results, and the analysis presented in the report reflect our own understanding and decisions. We take full responsibility for the correctness, integrity, and originality of the work submitted.
